2 Vowissen

2.1 Gradient

Der Gradient ist der Vektor aller partiellen Ableitungen einer Funktion \(f\).

Notation: \(\nabla_xf(x)\)

2.1.1 Beispiel

\(f(x, y) = 2x^2 + y^3\)

\(\rightarrow \nabla_xf(x)=(\begin{array}{c} f'_x \\ f'_y \end{array})= (\begin{array}{c} 4x \\ 3y \end{array})\)
